# Сайт детского сада № 30 (Самара)

Современный веб-сайт для МБДОУ "Детский сад комбинированного вида № 30" городского округа Самара.

## 🏗️ Архитектура

Проект построен по методологии **FSD (Feature-Sliced Design)** - архитектурной методологии для построения фронтенд приложений.

### Структура слоёв:

```
src/
├── app/          # Инициализация приложения
├── pages/        # Страницы приложения
├── widgets/      # Крупные самостоятельные блоки
└── shared/       # Переиспользуемый код
```

Подробнее см. [FSD_STRUCTURE.md](./FSD_STRUCTURE.md)

## 🎨 Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Tailwind CSS v4** - стилизация
- **Vite** - сборщик
- **Lucide React** - иконки
- **FSD** - архитектура

## 📦 Виджеты

- **Header** - Шапка сайта с навигацией
- **Hero** - Главный баннер с информацией о детском саде
- **SectionCards** - Карточки разделов сайта
- **ImportantInfo** - Важная информация для родителей
- **WelcomePoem** - Стихотворение о детском саде
- **QuickLinks** - Быстрые ссылки на популярные разделы
- **InfoBanner** - Баннеры с опросами и важной информацией
- **Footer** - Подвал сайта с контактами

## 🚀 Установка и запуск

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера (Vite dev server уже запущен в окружении Figma Make)
# Не требуется запускать вручную

# Сборка проекта (НЕ используется в Figma Make)
# pnpm build
```
